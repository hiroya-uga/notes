## マキネッタでの基本レシピ

| 手順 | 内容 | 備考 |
| :-: | - | - |
| 1 | C3Sの挽き目を**8**にする | 中煎り想定。 |
| 2 | **15g**のコーヒー豆を挽く | 3cups想定。 |
| 3 | マキネッタに水と粉をセットする | かならずドージングファンネルを使う。これを使うだけでちゃんとなる気がする。コーヒー豆がいつもより少なそうに見えても15g測ったことを信じる。 |
| 4 | 五徳サポートとマキネッタをセットして、火にかける | 火がマキネッタより外に出ないくらい弱火[^マキネッタの火加減]。 |
| 5 | ぐつぐつ音が始まった瞬間に火を止める | |
| 6 | すぐカップに注いで、すこし混ぜる。 | |

[^マキネッタの火加減]: 理想は2分半以内に抽出が終わること、かつ火がマキネッタの底面からはみ出さないことらしい。環境によってどう足掻いても2分半で終わらない場合は、お湯から始めるケースもあるらしいが、すぐに本体に熱が伝わってさわれなくなるためマキネッタ上部をしっかり絞めるにはミトンが必須そう。
