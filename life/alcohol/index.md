## 作ったカクテルのレシピ

### アマレットミルク

![アマレットミルクの写真](./cocktail-amaretto-milk.webp?size=400x400)

- ディサローノ アマレット： 30ml
- 牛乳： 120ml
- 氷： 適量

| 手順 | 内容 |
| - | - |
| 1 | グラスに氷を入れる |
| 2 | アマレットを注ぐ |
| 3 | 冷たい牛乳を注ぎ、軽くステアして完成 |

![アマレットオレンジの写真](./cocktail-amaretto-orange.webp?size=400x400)

- ディサローノ アマレット： 30ml

| 手順 | 内容 |
| - | - |
| 1 | グラスに氷を入れる |
| 2 | アマレットを注ぐ |
| 3 | 冷たいオレンジジュースを注ぎ、軽くステアして完成 |
