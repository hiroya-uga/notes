## 初期設定

### 1. 環境設定

- macOS：`Command` + `K`
- Windows：`Ctrl` + `K`

![PhotoShop環境設定画面のスクリーンショット](./photoshop-preference.png?size=1200x740)

| 手順 | 作業内容 | 備考 |
| :- | :- | :- |
| 1 | 「単位・定規」を開き、「単位」の「定規」と「文字」を「pixel」に変更 |  |
| 2 | 「ガイド・グリッド・スライス」を開き、「グリッド線」を8ピクセル、「分割数」を1に変更 | 8pxグリッドで作業する場合 |

### 2. カラー設定

- macOS：`Shift` + `Command` + `K`
- Windows：`Shift` + `Ctrl` + `K`

![PhotoShopカラー設定画面のスクリーンショット](./photoshop-color.png?size=1200x647)

| 手順 | 作業内容 | 備考 |
| :- | :- | :- |
| 1 | 「カラー設定」を開き、「設定」を「Web・インターネット用-日本」に変更 | |
