## やじるしは2回キーを押すだけで入力できる

日本語入力中に`z`キーに続けて`h` `j` `k` `l`キーのいずれかを押すと、やじるしが入力されます[^1]。

| 入力 | 出力 |
| :- | :- |
| `z` + `h` | ← |
| `z` + `j` | ↓ |
| `z` + `k` | ↑ |
| `z` + `l` | → |

[^1]: `vim`のカーソル移動キーと同じ並び。Google日本語入力やATOKなど主要な日本語IMEでも同じ入力で変換できるらしい。
