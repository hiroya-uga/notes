## JavaScriptは `[ ] ( ) ! +` の6つの記号だけでコードが書ける

文字列化した`"true"` `"false"` `"undefined"` `"[object Object]"`などからアルファベットを拾って、`eval()`する、というテクニック（？）です[^1]。
