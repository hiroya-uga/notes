## 1. 環境設定

- macOS：`Command` + `,`
- Windows：`Ctrl` + `,`

![Slack環境設定画面のスクリーンショット](./slack-setup.webp?size=1200x800)

| 手順 | 画面遷移 | 作業内容 | 備考 |
| :- | :- | :- | :- |
| 1 | 「通知」を開く |「マイキーワード」欄に通知してほしい単語をカンマ区切りで入力 | 自分の名前、担当プロジェクト名、関心のあるトピックなど |
| 2 | 「メッセージ & メディア」→「ワンクリック絵文字リアクション」 |「自分がよく使う絵文字」を選択 |  |
| 3 | 「詳細設定」→「メッセージ作成画面で `Enter` を押した時」 | 「改行する」に変更 |  |

## 2. リマインダーBotを使いたい

```slack
/remind list
```

`/remind`コマンドで自分や他のユーザー、チャンネルに対してリマインダーを設定できます。

コマンド生成には[Slackリマインダーコマンドジェネレーター](https://uga.dev/tools/slack-reminder-command-generator/)をご活用ください。
